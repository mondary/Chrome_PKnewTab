# Mondary NEW TAB

Chrome extension that replaces the New Tab page with the latest posts from `mondary.design` in a mosaic layout.

## Changelog

### 1.0.1 - 2026-04-10
- Fixed missing article images by switching to WordPress REST API featured media retrieval.
- Added robust image fallback handling when a featured image is unavailable.
- Kept RSS as fallback source if API fetch fails.

### 1.0.0
- Initial release.
